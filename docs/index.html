rust-resume (fr-rs)

Fast fuzzy finder TUI for coding agent session history. Search and resume sessions across 10 coding agents.

Install

curl -fsSL https://rucnyz.github.io/rust-resume/install.sh | bash

Or build from source:

cargo install --git https://github.com/rucnyz/rust-resume

Supported Agents

Claude Code, Codex CLI, GitHub Copilot CLI, Copilot VSCode, Crush, Gemini CLI, Kimi CLI, OpenCode, Qwen Code, Vibe

Keybindings

Navigation

KeyAction
↑ / kMove up
↓ / jMove down
PgUp / PgDnPage up / down in results
Ctrl+U / Ctrl+DScroll preview up / down
Home / EndJump to first / last result

Actions

KeyAction
EnterResume selected session
cCopy resume command to clipboard
Tab / Shift+TabCycle agent filter
Ctrl+SToggle sort (relevance / time)
Ctrl+`Toggle preview panel
Ctrl+PToggle preview layout (bottom / right)
Ctrl+EToggle mouse capture (on = scroll/click, off = select text)
Esc / Ctrl+CQuit

Search Input

KeyAction
Ctrl+AMove cursor to start
Ctrl+KDelete to end of line
Ctrl+WDelete word backward
BackspaceDelete character

Search Syntax

PrefixExampleMeaning
agent:agent:claudeFilter by agent
-agent:-agent:opencodeExclude agent
dir:dir:rust-resumeFilter by directory
date:date:today, date:3d, date:1wFilter by time range

CLI Flags

fr-rs                          # Open TUI
fr-rs --list                   # List sessions to stdout
fr-rs --list 'niri'            # Search and list
fr-rs --agent claude --list    # Filter by agent
fr-rs --rebuild --list         # Force rebuild index
fr-rs --stats                  # Show index stats
