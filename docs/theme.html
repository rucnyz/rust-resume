ase Integrations Theme

Material You Theme

ase uses Material You color roles for its TUI theme. All colors are configurable via the [theme] section in ~/.config/agents-sesame/config.toml.

Color Roles

RoleControlsDefault
primary Focused borders, title bar brand, footer key highlights, search box border #E87B35
on_surface Normal text, query text, active sort headers, title column text White
on_surface_variant Dim text, inactive borders, metadata (directory, turns), search prompt, placeholders, scrollbar thumb DarkGray
surface_variant Selected row background #28283C
surface_container Scrollbar track #3C3C3C
secondary User message prefix (»), project scope label Cyan
tertiary Local scope label, loading indicator, status messages #64FF64
primary_container Search highlight match background (reserved) Yellow
error Error messages (reserved) Red

Example Config

[theme]
primary = "#E87B35"
on_surface = "#FFFFFF"
on_surface_variant = "#808080"
surface_variant = "#28283C"
surface_container = "#3C3C3C"
secondary = "#64C8FF"
tertiary = "#64FF64"
primary_container = "#FFFF00"
error = "#FF0000"

All fields are optional. Omitted fields use the built-in defaults shown above. Values must be 6-digit hex colors with or without # prefix.

What's Not Themed

These colors are intentionally hardcoded and not part of the theme system:

  • Age gradient (date column): green → yellow → orange → gray based on session age. These are data visualization colors.
  • Per-agent colors: Each agent (Claude, Codex, etc.) has a brand color defined in the agent registry. These are identity colors, not theme colors.

Auto-Generate with matugen

Use matugen to generate a Material You theme from your wallpaper. See the Integrations page for setup instructions.
