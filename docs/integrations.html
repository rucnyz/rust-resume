ase Integrations Theme

Integrations

ase exposes scriptable CLI flags (--preview, --resume, --format) that make it easy to integrate with external tools.

fzf

Pipe session list into fzf with preview and resume-on-enter:

ase --list --format=tsv | fzf --delimiter='\t' --with-nth=2,3,4,5,6 \
  --preview='ase --preview {1}' \
  --bind='enter:become(ase --resume {1})'

How it works

FlagPurpose
--format=tsvTab-delimited output: ID, Agent, Title, Directory, Turns, Date
--with-nth=2,3,4,5,6Hide the ID column (field 1) from display, show Agent through Date
--preview='ase --preview {1}'Show full session content in fzf preview pane (field 1 = session ID)
--bind='enter:become(...)'On Enter, replace fzf process with ase --resume to resume the session

Add to your shell config as a function or alias:

# fish
function frs
  ase --list --format=tsv | fzf --delimiter='\t' --with-nth=2,3,4,5,6 \
    --preview='ase --preview {1}' \
    --bind='enter:become(ase --resume {1})'
end

# bash/zsh
frs() {
  ase --list --format=tsv | fzf --delimiter='\t' --with-nth=2,3,4,5,6 \
    --preview='ase --preview {1}' \
    --bind='enter:become(ase --resume {1})'
}

television

television is a fast TUI fuzzy finder. ase ships a cable channel config for it.

Setup

Save the following as ~/.config/television/cable/ase.toml (source):

[[cable_channel]]
name = "ase"
source_command = "ase --list --format=tsv"
preview_command = "ase --preview {0}"
preview_delimiter = "\t"

Launch:

tv ase

matugen

matugen generates Material You color schemes from wallpaper images. ase ships a template that auto-generates a theme config.

Setup

Save the following as ~/.config/matugen/templates/ase.toml (source):

[theme]
primary = "{{colors.primary.default.hex}}"
on_surface = "{{colors.on_surface.default.hex}}"
on_surface_variant = "{{colors.on_surface_variant.default.hex}}"
surface_variant = "{{colors.surface_variant.default.hex}}"
surface_container = "{{colors.surface_container.default.hex}}"
secondary = "{{colors.secondary.default.hex}}"
tertiary = "{{colors.tertiary.default.hex}}"
primary_container = "{{colors.primary_container.default.hex}}"
error = "{{colors.error.default.hex}}"

Add to your matugen config (~/.config/matugen/config.toml):

[templates.ase]
input_path = "~/.config/matugen/templates/ase.toml"
output_path = "~/.config/agents-sesame/config.toml"

Generate theme from a wallpaper:

matugen image /path/to/wallpaper.jpg

This writes a [theme] section to ~/.config/agents-sesame/config.toml. ase picks it up on next launch. See the Theme page for color role details.

Pipeline

wallpaper.jpg → matugen → config.toml [theme] section → ase reads on startup

JSON + jq

Use --format=json for structured output (one JSON object per line):

# Count sessions per agent
ase --list --format=json | jq -s 'group_by(.agent) | map({agent: .[0].agent, count: length})'

# Find sessions with most turns
ase --list --format=json | jq -s 'sort_by(-.turns) | .[0:5]'

# Get directories of recent Claude sessions
ase --list --agent claude --format=json | jq -r '.directory' | head -10
